村上の前帝の御時に

2016/1/1

村上の前帝の御時に 雪のいみじうふりたりけるを 様器にもらせ給ひて、梅の花をさして 月のいとあかきに 村上のみかど「これに歌よめ。いかゞいふべき」と兵衞の藏人に給はせたりければ、「雪月花の時」と奏したりけるをこそ、いみじうめでさせ給けれ。（宮）「歌などよむはよの常なり。かくおりにあひたる事なんいひがたき」とぞおほせられける。
